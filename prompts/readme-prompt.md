# README Prompt

This is the prompt used to generate the first draft of the project README. The final README was edited and tweaked by hand after generation.

**Model used:** Claude Opus 4.6 (Anthropic) via Claude Code CLI

---

## Prompt

```
Help me write a README for my class project. Use GitHub markdown. Keep it clean and not too long. It's a personal homepage with 3 pages (index, interests, resume).

Sections I need:
- Project name and short description
- Live demo link (placeholder)
- Video demo link (placeholder)
- About the project
- List of pages
- Tech stack
- Install and run instructions
- Author info (Khush Patel)
- Class info (CS5610 Web Development at Northeastern, instructor John Alexis Guerra Gómez)
- GenAI disclosure section
- License section
-Contributing section

Tech used: HTML, CSS, Bootstrap 5, vanilla JS. Google Fonts. Reload for dev server.

```
